# Bootstrap 

## Alignment
```html
<div class="text-left">I'm left-aligned</div>
<div class="text-center">I'm center-aligned</div>
<div class="text-right">I'm right-aligned</div>
<div class="text-justify">I'm justified</div>
```

## Border Radius
```html
<img src="..." alt="..." class="rounded">
<img src="..." alt="..." class="rounded-top">
<img src="..." alt="..." class="rounded-right">
<img src="..." alt="..." class="rounded-bottom">
<img src="..." alt="..." class="rounded-left">
<img src="..." alt="..." class="rounded-circle">
<img src="..." alt="..." class="rounded-pill">
<img src="..." alt="..." class="rounded-0">
```

## Display
```html
<span class="d-block">I'm block!</div>
<div class="d-inline">I'm inline!</div>
<div class="d-inline-block">I'm inline-block!</div>
<div class="d-flex">I'm flex!</div>
<div class="d-none">I'm hidden</div>
```
```html
<div class="d-none d-md-block">I'm visible on md+ screens</div>
```

## Float
```html
<div class="float-left">Floating left</div>
<div class="float-right">Floating right</div>
<div class="float-none">Not floating</div>
```

## Shadows
```html
<div class="shadow-none">No shadow</div>
<div class="shadow">A shadow</div>
<div class="shadow-sm">A smaller shadow</div>
<div class="shadow-lg">A larger shadow</div>
```

## Visibility
```html
<div class="visible">...</div>
<div class="invisible">...</div>
```
